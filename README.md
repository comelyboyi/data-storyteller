# AI数据故事家（Data Storyteller）

## 项目简介

AI数据故事家是一款面向"数据智能分析与应用"赛道的数据叙事技能。它将传统的"数据分析"升级为"数据叙事"——不仅告诉你数据是什么，更告诉你数据意味着什么、应该怎么做。

## 核心理念

大多数数据分析工具输出的是图表和统计表。决策者看到的是一堆数字，却不知道该做什么。AI数据故事家解决的正是这个问题：它将数据转化为一个有开头、有冲突、有结论、有行动建议的完整故事，让数据真正驱动决策。

## 工作流程

```
原始数据 → 数据画像 → 模式发现 → 故事构建 → 行动洞察
```

1. **数据画像**：自动识别数据结构、类型、质量，建立数据的基本认知。
2. **模式发现**：从数据中识别趋势、异常、相关性、分布和对比五类模式。
3. **故事构建**：将发现的模式编织成"开篇→主体→转折→结尾"的连贯叙事。
4. **行动洞察**：提炼核心结论、关键指标和具体行动建议。

## 使用方式

将以下内容作为技能安装后，上传Excel/CSV文件或粘贴数据表格即可触发：

- 上传数据文件
- 粘贴数据表格
- 使用触发短语："分析这份数据"、"帮我解读数据"、"生成数据故事"

## 输出说明

技能输出包含四个部分：

| 部分 | 内容 | 用途 |
|------|------|------|
| 数据画像 | 数据集基本描述和质量评估 | 了解数据全貌 |
| 数据故事 | 500-800字的叙事文本 | 理解数据含义 |
| 关键发现 | 3-5个带数值的关键发现 | 快速掌握要点 |
| 行动建议 | 2-4条具体可执行的建议 | 指导决策行动 |

## 适用场景

- 销售数据分析：季度业绩回顾、区域对比、产品表现
- 用户行为分析：用户留存、转化漏斗、行为路径
- 财务报告分析：收入构成、成本结构、利润趋势
- 问卷调查分析：满意度调查、市场调研、用户反馈
